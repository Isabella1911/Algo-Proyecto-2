# Algo-Proyecto-2
Rod Cutting Problem


Implementaciones:
  1. Programación Dinámica - Bottom-Up  O(n^2)
  2. Programación Dinámica - Top-Down (Memoización)  O(n^2)
  3. Greedy (ratio precio/longitud)  O(n log n)

Incluye:
  - Análisis empírico con múltiples tamaños de entrada
  - Diagrama de dispersión de tiempos de ejecución
  - Regresión polinomial ajustada a los datos
